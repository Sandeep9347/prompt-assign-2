# 1. Summarization

Naive Prompt:

Summarize the text below.
“The Industrial Revolution began in the 18th century... [text continues]”

Refined Prompt:

Summarize the following passage in 3–4 concise sentences, focusing on key causes, effects, and significance.
Avoid minor details and use neutral, academic tone.

Text: “The Industrial Revolution began in the 18th century...”

# 2. Question and Answering

Naive Prompt:

What is photosynthesis?

Refined Prompt:

You are a biology tutor.
Explain what photosynthesis is, including the chemical process, organisms involved, and its importance to ecosystems.
Keep the explanation simple and suitable for high school students.

# 3. Content Writing

Naive Prompt:

Write an article about healthy eating.

Refined Prompt:

Write a 700-word blog article on healthy eating habits for office workers.
Include a catchy introduction, three main sections (meal planning, snacks, hydration), and a motivational closing paragraph.
Use a friendly, engaging tone and SEO-friendly keywords like “healthy office meals” and “nutrition tips”.

# 4. Sentiment Analysis / Classification

Naive Prompt:

What is the sentiment of this text: “I love this product, it’s amazing!”?

Refined Prompt:

Analyze the sentiment of the following text and classify it as Positive, Negative, or Neutral.
Also, provide a brief reason for your classification.

Text: “I love this product, it’s amazing!”

# 5. Code Generation

Naive Prompt:

Write a Python program for bubble sort.

Refined Prompt:

Write a Python program that implements the Bubble Sort algorithm.
The program should:

Accept a list of integers as input

Sort them in ascending order

Print both the unsorted and sorted lists
Include comments explaining each major step.
